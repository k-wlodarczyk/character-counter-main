Character Counter - docs

Characters

As characters are counted all kinds of signs - both visible for users (e.g. letters or numbers) and not visible (e.g. new lines or tabs)

If option "Exclude spaces" is checked, spaces are ignored in characters counting.

Examples of characters (separated by semicolon):
a ; 3 ; enter ; space ; #

Words

As words are counted Letters, numbers, asterisks, underscores and polish characters. One character is enough to make a new word. Any other characters than indicated before is a separator for words.

Examples of words:

a b => 2 words

word1-word2 => 2 words

wo*d => 1 word

* _ => 2 words

@#%^ => 0 words

Sentences

As characters are counted sets with at least one word. Sentences are separated by characters: . ! ? ...

Sentence is counted if there is at least one word started after one separator (or beginning of text) and is finished before another separator (or end of text)

Examples of sentences:

Sentece 1. Sentence 2. => 2 sentences

sentence 1? sentence 2! sentence3...sentence 4 => 4 sentences

_. sentence2 #@%. => 2 sentences
